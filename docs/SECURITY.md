# 安全漏洞处理策略 (Security Policy)

为了保障闫士博个人主页的运行安全以及防范潜在的网络攻击或数据泄露，我们建立了本安全漏洞上报机制。

---

## 1. 支持的版本 (Supported Versions)

我们仅对目前在线上生产环境中运行的 `main` 分支代码提供安全更新：

| 版本层级 | 支持状态 |
| :--- | :--- |
| main (生产分支) | ✅ 支持 (Active Support) |
| 其他开发分支 | ❌ 不支持 (No Support) |

---
