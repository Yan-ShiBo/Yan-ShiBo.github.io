# Yan-ShiBo.github.io

这是闫士博个人主页的重构版本，适用于直接部署到 GitHub Pages。

## 目录说明

- `index.html`：主页
- `introduction.html`：个人档案
- `research.html`：研究方向
- `projects.html`：项目展示
- `resume.html`：合并后的简历页（在线内容 + PDF 简历 + 论文 + 本科成绩单）
- `stats.html`：访问统计
- `en/`：英文独立页面
- `docs/`：论文、简历、本科成绩单 PDF
- `img/`：证明材料、项目获奖照片、论文/成绩单预览图
- `resource/css/site.css`：全站样式
- `resource/js/site.js`：主题、导航、返回顶部、图片放大
- `resource/js/stats.js`：公开统计与本地统计逻辑

## 部署说明

直接将仓库内容推送到 GitHub Pages 对应分支即可。
如果你只想覆盖线上站点，把本目录全部内容上传到站点根目录即可。
